# Student Registration Form (AngularJS)

A robust student registration form built using the AngularJS framework. This project demonstrates form validation, two-way data binding, and state management in a single-page application context.

## Features

- **Real-time Validation**: Validates name, email, age, and course selection as the user types.
- **Age Restriction**: Specifically restricts registration to students between the ages of 18 and 30.
- **Data Binding**: Uses `ng-model` for seamless data synchronization between the UI and the controller.
- **Submission Summary**: Displays the successfully registered student's details upon form submission.
- **Form Reset**: Automatically clears and resets the form state after a successful registration.

## Technologies Used

- **AngularJS (v1.8.2)**: Core framework for logic and data binding.
- **HTML5/CSS3**: For semantic structure and responsive styling.

## Getting Started

1. **Recommended**: Right-click `index.html` and select **"Open with Live Server"** (if using VS Code) to ensure all AngularJS components load correctly.
2. Alternatively, open `index.html` directly in your web browser.
3. Fill out the registration form and observe the real-time validation.
4. Click "Register" to see the submitted data displayed below the form.
