# RDF Book Search

A lightweight web application that demonstrates basic RDF (Resource Description Framework) data parsing using JavaScript. It allows users to search for book authors based on titles defined in a Turtle-like RDF format.

## Features

- **RDF Parsing**: Demonstrates how to parse raw RDF strings in Turtle format using string manipulation and regex.
- **Book Author Search**: Search for authors of popular books like "Harry Potter" or "The Hobbit".
- **Dynamic Results**: Instantly displays the author's name or a "Not found" message.

## Technologies Used

- **HTML5**: For the search interface.
- **CSS3**: For basic styling and layout.
- **JavaScript (Vanilla)**: For the RDF parsing logic and DOM manipulation.

## How to Use

1. **Recommended**: Right-click `index.html` and select **"Open with Live Server"** (if using VS Code).
2. Alternatively, open `index.html` directly in any modern web browser.
3. Search for authors of popular books like "Harry Potter" or "The Hobbit".
