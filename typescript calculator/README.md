# Interactive Calculator

A clean and functional web-based calculator designed for basic arithmetic operations. This project focuses on a responsive user interface and straightforward logic.

## Features

- **Basic Operations**: Support for Addition, Subtraction, Multiplication, and Division.
- **History Tracking**: Displays the last calculated expression for easy reference.
- **Error Handling**: Gracefully handles invalid mathematical expressions by displaying an "Error" message.
- **Clear Functionality**: One-click reset for both the display and calculation history.
- **Responsive UI**: A modern, centered layout with distinct styling for operators and numbers.

## Technologies Used

- **HTML5**: For the calculator structure.
- **CSS3**: For the layout and interactive button designs.
- **JavaScript**: For the calculation logic (using `eval()` for expression parsing).

## How to Use

1. **Recommended**: Right-click `index.html` and select **"Open with Live Server"** (if using VS Code).
2. Alternatively, open `index.html` directly in your browser.
3. Use the on-screen buttons to enter numbers and operators.
4. Click the `=` button to see the result and click `C` to clear.
