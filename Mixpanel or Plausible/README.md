# Startup Analytics Hub

A simple web application demonstrating how to track user activity and page views using `localStorage`. This project serves as a conceptual comparison between popular analytics tools like Mixpanel, Plausible, and Google Analytics.

## Features

- **Page View Tracking**: Automatically tracks visits to Home, Tools, and Reports pages.
- **Total Visit Counter**: Keeps track of the cumulative number of visits to the hub.
- **Analytics Dashboard**: A dedicated reports page to visualize the tracked data.
- **Responsive Design**: Clean and modern UI with a focus on usability.

## Files

- `index.html`: The landing page of the Analytics Hub.
- `tools.html`: Information about various analytics tools.
- `reports.html`: Displays the tracked page view statistics.
- `script.js`: Contains the logic for tracking and reporting using `localStorage`.
- `style.css`: Modern styling for the application.

## How to Use

1. **Recommended**: Right-click `index.html` and select **"Open with Live Server"** (if using VS Code) for the best experience.
2. Alternatively, open `index.html` directly in any web browser.
3. Navigate through the different pages (Home, Tools, Reports).
4. Check the **Reports** page to see your real-time browsing statistics stored in your browser's local storage.
