# WebX - Web Development Exercises & Projects

Welcome to **WebX**, a collection of various web development projects, exercises, and demonstrations. This repository covers a wide range of technologies, from basic HTML/CSS/JS to advanced Node.js, Express, TypeScript, and AngularJS.

## Project Directory

This workspace contains the following projects:

### 🌐 Frontend & Frameworks
- **[angular form](./angular%20form)**: A student registration form built with AngularJS featuring real-time validation.
- **[typescript calculator](./typescript%20calculator)**: An interactive calculator demonstrating basic UI logic.
- **[typescript student_management](./typescript%20student_management)**: A management system built with TypeScript.
- **[Mixpanel or Plausible](./Mixpanel%20or%20Plausible)**: A startup analytics hub demonstrating `localStorage` tracking.

### 🚀 Backend & Node.js
- **[built-in modules](./built-in%20modules)**: A native Node.js server using only core modules (`http`, `fs`).
- **[express](./express)**: Web application demonstrations using the Express.js framework.
- **[restful api](./restful%20api)**: A project showcasing the implementation of RESTful architectural principles.
- **[stream-buffer](./stream-buffer)**: Demonstrations of Node.js Streams and Buffers for efficient data handling.
- **[Full stack](./Full%20stack)**: A comprehensive project integrating both frontend and backend components.

### 📊 Specialized Demos
- **[Rdf and sparql](./Rdf%20and%20sparql)**: A demonstration of RDF data parsing and searching using JavaScript.

## Getting Started

### 🌐 Frontend Projects
For projects in the **Frontend & Frameworks** and **Specialized Demos** sections:
1. Open the project folder in VS Code.
2. Right-click `index.html` and select **"Open with Live Server"**.
3. *Note: Opening the file directly in the browser also works, but Live Server is recommended for features like AngularJS or TypeScript compilation.*

### 🚀 Backend Projects
For projects in the **Backend & Node.js** section:
1. Ensure [Node.js](https://nodejs.org/) is installed.
2. Open your terminal in the specific project directory.
3. Run the server command (usually `node server.js`).
